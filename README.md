# chenyu-opinion

基于[辰语](https://chenyuweb.com/)网站观点文章的 Kiro CLI skill，输出有深度的中文社会观点分析。

  辰语（chenyuweb.com）是一个独立中文内容平台，核心理念是**"每一个声音都值得被听见"**，现有 3285
  篇文章，覆盖社会民生、经济制度、权力司法、AI科技、性别婚姻、教育个体等议题。


  本项目将辰语的文章视角结构化为一个观点输出助手，针对用户提出的话题，输出有立场、有温度、关注底层与边缘群体的深度中文分
  析。

  功能

  输入任意社会话题，助手将按以下结构输出：

  1. 核心立场 — 一句话点明观点
  2. 论据展开 — 2-3 个角度，结合辰语文章视角
  3. 底层逻辑 — 指向结构性问题（制度、分配、权力、个体处境）

  文章库覆盖范围
  
│    分类    │            代表议题            │
│ 社会与民生 │ 打工人处境、教育内卷、医疗悲剧 │
│ 经济与制度 │ 养老金分配、灵活就业、数据造假 │
│ 权力与司法 │ 司法腐败、企业权力、基层抗争   │
│ AI 与科技  │ AI泡沫、大厂风险、智能危机     │
│ 性别与婚姻 │ 东亚婚姻困局、厌女文化         |
│ 教育与个体 │ 考编焦虑、历史教育、网红祛魅   │


  观点风格

  有立场、有温度，批判权力傲慢，关注底层与边缘群体。

  数据来源

  全部观点基于辰语公开文章，原文见 https://chenyuweb.com/

  
## 安装

```bash
npx skills add chenyuwebwawa/chenyu-opinion-skill
```

## 使用

```
/chenyu-opinion 如何看待xxxx？
```
